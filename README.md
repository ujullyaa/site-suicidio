# 📊 Observatório de Dados e Apoio à Prevenção do Suicídio

Este projeto é uma **plataforma de conscientização e visualização de dados** relacionada à prevenção do suicídio, desenvolvida pelo Laboratório de Neurociência Translacional da UFSC.

---

## 🎯 Propósito do projeto

O sistema tem três pilares principais:

---

### 1. 📊 Ciência e dados (Observatório)

Foco na análise e visualização de dados reais:

- Evolução dos casos ao longo dos anos
- Diferenças por gênero
- Faixas etárias mais afetadas
- Identificação de padrões e tendências

👉 Essa camada é alimentada por dados CSV processados no backend e exibidos em gráficos no frontend.

---


### 3. 🎓 Contexto acadêmico (UFSC)

Projeto desenvolvido no ambiente acadêmico:

- Discentes da UFSC
- Vinculado a atividades de pesquisa e extensão
- Uso de dados públicos
- Finalidade educativa e científica

---

## 🧠 Resumo

Este projeto utiliza tecnologia e dados reais para ajudar a compreender um problema social sério, ao mesmo tempo em que promove informação confiável e apoio emocional.

---

## ⚙️ Tecnologias utilizadas

- Node.js
- Express
- TypeScript
- csv-parser
- React (frontend)
- Recharts / Chart.js (gráficos)

---

## 📊 Funcionalidades

- Leitura de CSV
- Processamento de grandes volumes de dados
- Agrupamento por ano
- API para consumo de gráficos
- Visualização em dashboards

---
